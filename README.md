# Servicio Técnico Computación 1234

Sitio web oficial de **Servicio Técnico Computación 1234**, un servicio técnico especializado en reparación, mantenimiento y soporte de equipos informáticos en Santiago de Chile.

### 🌐 Vista previa del proyecto
Un sitio web moderno, responsive y fácil de navegar para presentar los servicios de reparación de computadores, notebooks, eliminación de virus, mantenimiento preventivo, armado de PCs y más.

### ✨ Características

- **Diseño responsive** (adaptado a móviles, tablets y escritorio)
- **Navegación intuitiva** con menú fijo
- Secciones completas:
  - Inicio con hero atractivo
  - Sobre Nosotros
  - Servicios detallados
  - Portafolio de trabajos realizados
  - Página de contacto
- Optimizado para SEO básico
- Código limpio y bien estructurado (HTML5 + CSS3)

### 🛠️ Tecnologías utilizadas
- CSS3
- JavaScript (ligero)
- Diseño responsive con Flexbox y Grid

### 📸 Imágenes incluidas
- Logo: `edubecam.png`
- Imágenes de servicios y portafolio
- Fotos hero
